# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in Grid Watcher, please DO NOT open a public issue.
Instead, please email the author directly at **zuudevs@gmail.com**.

We will review the issue and publish a fix as soon as possible.
